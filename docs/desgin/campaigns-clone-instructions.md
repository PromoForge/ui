# Campaigns Page — Page Structure & Clone Instructions

## Navigation & Header

- The page lives under: **Apps → KSA Production → Campaigns** (breadcrumb at top).
- A **"+ Create Campaign"** blue button sits in the left sidebar, above the navigation menu.
- A **"Manage Campaign Data"** dropdown button is in the top-right corner of the main content area.

---

## Three Tabs

The campaign list has exactly **three tabs** at the top of the content area:

| Tab | Description |
|---|---|
| **Current** | The default active tab (underlined in blue). Shows all active/live campaigns. |
| **Calendar View** | A visual calendar display of campaigns. |
| **Archived** | Shows archived/inactive campaigns. |

---

## Filters & Search (within the Current tab)

- A **search bar** labelled "Campaign name, tags or id" for quick lookup.
- A **Filter** button (top right of the list).
- **Status filter pills** you can click to narrow results:
  - 🟢 Scheduled (0)
  - 🟢 Running (182)
  - 🔴 Expired (661)
  - 🟡 Disabled (1132)
  - Low on budget (4)
  - Expiring soon (2)
- **Pagination**: campaigns are shown 50 at a time (e.g. "1 – 50"), with left/right arrows to navigate pages.

---

## Campaign Table Columns

Each row in the table contains: **Campaign Name**, **Type** (icon), **ID**, **Store**, **Start**, **End**, **Coupon Redemptions**, **Total Discounts**, **Last Activity**, **Created**, **Created By**, **Copy**, **Template**.

> Note: Not all columns are visible simultaneously — the table scrolls horizontally. The last two columns are always the action columns.

---

## The Two Action Icons per Campaign Row (rightmost columns)

1. **Copy** (duplicate icon) — tooltip reads **"Copy campaign"**. This is the clone button.
2. **Template** (layout/page icon) — saves the campaign as a template.

---

## How to Clone a Campaign

**Step-by-step instructions for an agent:**

1. **Navigate** to the Campaigns section by clicking **Campaigns** in the left sidebar under "Manage Application".

2. **Locate the campaign** you want to clone. Use the search bar (type the campaign name, tag, or ID) or use the status filter pills to narrow the list. Campaigns are paginated 50 per page — use the arrow buttons to navigate if needed.

3. **Find the "Copy" button** in the same row as the target campaign. It is the second-to-last column on the right side of the table, labeled **COPY** in the column header. The icon looks like two overlapping squares (a duplicate/copy icon). Hovering over it reveals the tooltip **"Copy campaign"**.

4. **Click the Copy icon** for the desired campaign row. This will trigger the clone action — do not click the campaign name link itself (that opens the campaign detail), and do not click the Template icon next to it (that saves it as a template, which is a different action).

5. **Wait for confirmation** — after clicking, a dialog or new campaign draft is expected to be created. Follow any prompts that appear (e.g., naming the copied campaign).

> **Important distinction:** There are **two** action icons per row. The **Copy** icon (left of the pair) clones the campaign. The **Template** icon (right of the pair) saves it as a reusable template. Make sure to click the correct one.
